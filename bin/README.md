# boca_with_spark
Projeto de LP2


## Run Application 
####  Step 0
`cd <DIRECTORY HOME>`

####  Step 1
`mvn clean install`

####  Step 2
`mvn compile`

####  Step 3
`mvn package`

####  Step 4
`mvn exec:java -Dexec.mainClass="Main"`

#### Route: Hello World
`http://localhost:4567/hello`

## Stop
`stop();`
